歡迎使用 discord.py¶

discord.py 是一個現代、易於使用、功能豐富且支援異步的 Discord API 包裝器。

功能：

  • 使用 async/await 語法的現代 Pythonic API

  • 合理的速率限制處理機制，防止 429 錯誤

  • 用於協助機器人開發的命令擴充功能

  • 物件導向設計，易於使用

  • 在速度與記憶體使用上皆經過最佳化
