介紹¶

這是 discord.py 的文件，discord.py 是一個 Python 程式庫，用於協助建立使用 Discord API 的應用程式。

必要條件¶

discord.py 支援 Python 3.8 或更高版本。不提供對較舊 Python 版本的支援。不支援 Python 2.7 或更低版本。不支援 Python 3.7 或更低版本。

安裝¶

您可以直接從 PyPI 取得此程式庫：

python3 -m pip install -U discord.py

如果您使用的是 Windows，則應改用以下指令：

py -3 -m pip install -U discord.py

若要取得語音支援，您應使用 discord.py[voice] 而非 discord.py，例如：

python3 -m pip install -U discord.py[voice]

在 Linux 環境中，安裝語音支援需要取得以下依賴：

  • libffi

  • libnacl

  • python3-dev <https://packages.debian.org/python3-dev>`_

對於基於 Debian 的系統，以下命令可以取得這些依賴：

$ apt install libffi-dev libnacl-dev python3-dev

請記得檢查您的權限！

虛擬環境¶

有時您可能希望避免程式庫污染系統安裝，或需要使用與系統上已安裝版本不同的程式庫。您也可能沒有全系統安裝程式庫的權限。為此，Python 3.3 起標準庫提供了一個稱為「虛擬環境」的概念，來協助維護這些獨立的版本。

更深入的教學請參考 Virtual Environments and Packages。

不過，以下是快速上手的方法：

  1. 前往您的專案的工作目錄：

    $ cd your-bot-source
$ python3 -m venv bot-env

  2. 激活虛擬環境：

    $ source bot-env/bin/activate

    在 Windows 上，您可以這樣激活虛擬環境：

    $ bot-env\Scripts\activate.bat

  3. 像平常一樣使用 pip：

    $ pip install -U discord.py

恭喜！您現在已經設定好虛擬環境了。

備註

使用 py -3 執行的腳本會忽略當前已激活的虛擬環境，因為 -3 指定了全域範圍。

基礎概念¶

discord.py 的核心概念是 事件。事件是您監聽並回應的事物。例如，當有新訊息時，您會收到一個關於該訊息的事件，然後您可以對其做出回應。

以下是一個快速展示事件運作方式的範例：

# 此範例需要 'message_content' intent。

import discord

class MyClient(discord.Client):
    async def on_ready(self):
        print(f'Logged on as {self.user}!')

    async def on_message(self, message):
        print(f'Message from {message.author}: {message.content}')

intents = discord.Intents.default()
intents.message_content = True

client = MyClient(intents=intents)
client.run('my token goes here')
